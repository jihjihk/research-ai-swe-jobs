## How to read each claim page

Each page is structured as:

1. **Claim** — a complete sentence.
2. **Evidence** — numbers, figures with paths, sensitivities.
3. **What it rules out** — contradicted priors.
4. **Limitations** — known confounders.
5. **Links** — to raw reports in the audit trail.
