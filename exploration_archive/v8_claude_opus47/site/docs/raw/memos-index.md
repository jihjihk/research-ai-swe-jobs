## Reading order

If you are new to the project, read gate_0 first. It captures what a careful research advisor would commit to before touching the data — priors, methodology, pre-committed ablations. The rest of the memos trace how those priors held up.

## Post-mortem

Gate 0 got several things right:

- **Length growth as a confound** was identified before any data was queried. V1 confirmed it.
- **Asaniczka zero entry labels** was called out pre-dispatch. Every Wave 2+ claim respected the arshkon-only baseline.
- **Company concentration** was flagged. T06 identified 240 specialists; T16 decomposed 102% of the AI rise as within-company.

Gate 0 got several things wrong:

- **Priority on junior share.** Gate 0 treated junior as the dynamic cell. T12 + T21 showed seniors moved more.
- **Anticipatory AI over-specification.** Gate 0 expected employers to lead workers. T23 inverted the direction.
- **Seniority convergence.** Gate 0 hypothesized AI might blur the junior/senior gap. T20 sharpened it.

These misses are on the record in the memos. The paper should cite them as part of the methodological contribution.
