# T27: Static Exploration Evidence Package

## Scope

Built a static, navigable evidence package under `exploration/site/` using existing reports, memos, and figures only. No new data analysis, parquet inspection, preprocessing, or LLM calls were run.

## Outputs

| Output | Path |
|---|---|
| MkDocs source | `exploration/site/docs/` |
| MkDocs config | `exploration/site/mkdocs.yml` |
| MARP source deck | `exploration/site/slides/deck.md` |
| Built static site | `exploration/site/site/` |
| Slide HTML | `exploration/site/site/presentation.html` |
| Slide PDF | `exploration/site/site/presentation.pdf` |
| Raw copied reports | `exploration/site/docs/raw/reports/` |
| Raw copied memos | `exploration/site/docs/raw/memos/` |

## Build Status

Commands used from `exploration/site/`:

```bash
../../.venv/bin/python -m mkdocs build --strict
npx @marp-team/marp-cli slides/deck.md -o site/presentation.html --html
npx @marp-team/marp-cli slides/deck.md -o site/presentation.pdf --pdf --allow-local-files --browser chrome --browser-path ./scripts/chromium-no-sandbox --browser-timeout 120
```

MkDocs build completed successfully. MARP HTML export completed successfully. MARP PDF export completed after using a local Chromium wrapper with `--no-sandbox`; the first Firefox-based attempts timed out and a direct Chromium launch failed on sandbox restrictions.

Verification:

- `site/slides/index.html` embeds `../presentation.html`.
- `presentation.html` and `presentation.pdf` return HTTP 200 from the local server.
- PDF has 23 pages and includes the slide figure images.

## Hosting

Port 8080 was occupied by an existing Python server, so the package is hosted from `exploration/site/site/` on port 8081.

| URL type | URL |
|---|---|
| Local | `http://127.0.0.1:8081/` |
| Tailscale | `http://100.127.245.121:8081/` |

Server command:

```bash
python3 -m http.server 8081 --bind 0.0.0.0
```
