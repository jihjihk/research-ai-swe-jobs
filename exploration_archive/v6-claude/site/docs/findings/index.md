# Findings

The Wave 1-4 exploration produced four core findings, one methodological contribution, and a list of claims that did not survive robustness checks. Each page in this tab answers one question for a skeptical reader: *what is the claim, what evidence supports it, what sensitivity checks does it survive, and where do I read the raw reports?*

## Ranked by evidence strength

1. **[The AI explosion is field-wide](01-ai-explosion-field-wide.md)** — strong. 92% within-company, survives macro noise at 24.7×, 0-7% style attenuation, field-wide via DiD.
2. **[Posting language lags worker AI adoption by ~10×](02-posting-lags-workers.md)** — strong. Lead finding. Direction robust 50-85% worker-rate sensitivity band.
3. **[Senior orchestration specialization](03-senior-orchestration.md)** — strong. Co-headline. 100% pattern precision on 100-row audit.
4. **[Tool-stack restructuring and archetype pivot](04-toolstack-pivot.md)** — strong. 46-company adopter cluster plus 68% new-entrant LLM/GenAI wave. 74.6% archetype pivot rate.
5. **[What does not survive](05-what-does-not-survive.md)** — strong null on junior-narrowing, boundary sharpening, length-as-scope inflation.
6. **[Robustness framework (methodological contribution)](06-robustness-framework.md)** — novel combination of within-source calibration, specification dependence, macro ratios, and authorship matching.
