- **Software Engineer (General):** Core developers applying engineering to software design, development, and maintenance. *Definition Source:* SFIA “Software engineering practitioners”【49†L281-L289】. *Examples:* Software Engineer, Software Developer, SRE (if not separately classified)【49†L281-L289】【51†L75-L79】.  

- **Frontend/Web Engineer:** Focused on user-facing web/mobile interfaces and UI/UX. *Definition Source:* O*NET “Web Developers”【65†L67-L75】 (covers building websites/apps). *Examples:* Web Developer, Frontend Engineer, UI/UX Engineer, Full-Stack Web Developer【65†L67-L75】.  

- **Backend/API/Services Engineer:** Focused on server-side logic, APIs, databases, and services. *Source:* Implied by software-engineering taxonomy and industry usage【51†L75-L79】. *Examples:* Backend Engineer, API Developer, Microservices Engineer, Application Developer【51†L75-L79】.  

- **Mobile Engineer:** Specialist in native or cross-platform mobile app development. *Source:* Industry/job-posting practice (no single taxonomy). *Examples:* iOS Engineer, Android Developer, Mobile Application Engineer.  

- **Embedded/Systems Engineer:** Works on firmware, IoT, or hardware-near software. *Source:* SFIA lists “Real-time/embedded systems development” as a key skill under Software Eng【49†L297-L304】. *Examples:* Embedded Systems Engineer, Firmware Developer, IoT Engineer.  

- **Data Engineer / Data Platform:** Builds and maintains data pipelines, warehouses, and infrastructure. *Source:* SFIA “Data engineering practitioners”【47†L220-L229】. *Examples:* Data Engineer, ETL Engineer, Data Platform Engineer.  

- **Machine Learning (ML) Engineer:** Builds ML models and pipelines (pre-LLM). *Source:* SFIA “Data science practitioners” (ML skill set)【47†L190-L198】. *Examples:* ML Engineer, AI/ML Engineer (traditional), NLP Engineer (non-LLM).  

- **AI / LLM Engineer (Emergent):** Develops LLM-based/agent systems (prompt engineering, RAG). *Source:* Newly emerging in job postings (Indeed, LinkedIn trend reports); no formal taxonomy. *Examples:* AI Engineer, LLM Engineer, GenAI Engineer, Agent Engineer.  

- **DevOps / SRE / Platform Engineer:** Ensures CI/CD, infrastructure, and reliability. *Source:* SFIA “DevOps” role family【49†L354-L362】. *Examples:* DevOps Engineer, Site Reliability Engineer, Platform Engineer.  

- **Security Engineer:** Focuses on application or infrastructure security. *Source:* SFIA “Information and cyber security” role family【54†L496-L504】. *Examples:* Security Engineer, Application Security Engineer, Cybersecurity Engineer.  

- **QA / Test Engineer:** Focuses on software quality, testing, and validation. *Source:* SFIA “Testing” role family【54†L627-L635】. *Examples:* QA Engineer, Test Automation Engineer, QA/Test Lead.  

- **Solutions / Field / Customer Engineer:** Deploys and integrates solutions with customers. *Source:* Industry practice (SFIA “Solution Architecture” hints at end-to-end solutions【68†L13-L21】). *Examples:* Solutions Engineer, Field Applications Engineer, Customer Engineer, Pre-Sales Engineer.  

- **Legacy Technology Specialist:** Maintains older stacks (e.g. mainframes, .NET Framework). *Source:* Identified in industry discourse as roles in declining tech; no formal taxonomy. *Examples:* Mainframe Developer, COBOL Engineer, Legacy .NET Developer.  

- **Data / Analytics:** Focused on transforming raw data into usable analytical, operational, or decision-support outputs. Definition Source: SFIA “Data and analytics” role family, especially “Data science practitioners” and “Data engineering practitioners”; O*NET “Data Scientists” 15-2051.00. Examples: Data Scientist, Data Analyst, Analytics Engineer, Data Visualization Developer.
